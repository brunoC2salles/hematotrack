# Hematotrack — Landing Page

Landing page da plataforma Hematotrack. Self-contained (sem dependências externas além de fontes Google).

## Deploy no Vercel
1. Push para GitHub
2. Vercel → Import → Framework: **Other** → Output Directory: *(vazio)* → **Deploy**

## Estrutura
```
hematotrack-site/
├── index.html      ← página completa
├── vercel.json     ← headers de segurança
├── .gitignore
└── README.md
```
